
this file explain how to use DGtal_Pred

First of all you have to install DGtal if you don't i refer you to the official website of DGtal which is : http://libdgtal.org/
and follow their explication to install it on your computer.

  git clone https://github.com/SqLL/DGtal_PRED

You have to move or create a link of the file DGtalconfig.cmake.
or you have to export the variable DGtal.

  export DGtal /path/of/your/installation


and now it's should works

test it using

  cmake .

then 

  make
